Код внесён, теперь поставлю зависимости и проверю, что проект собирается без ошибок TypeScript.

Инициализировал проект из шаблона `https://github.com/irpsv/base-project-for-agent` и реализовал дашборд матрицы Эйзенхауэра для Bitrix24.

Что сделано:

- В `apps/server/src/index.ts` добавлен API-эндпоинт `GET /api/bitrix/tasks/eisenhower`.
- Эндпоинт запрашивает задачи через `tasks.task.list`, обходит пагинацию и раскладывает задачи по 4 квадрантам:
  - `important_urgent`
  - `important_not_urgent`
  - `not_important_urgent`
  - `not_important_not_urgent`
- Логика классификации:
  - **Важно**: по флагу `IMPORTANT` (и дополнительно по `PRIORITY > 0` как fallback)
  - **Срочно**: если у задачи заполнен `DEADLINE`
- В `apps/client/src/main.ts` полностью заменён UI на дашборд 2x2:
  - 4 карточки-квадранта матрицы
  - вывод задач с `ID`, дедлайном, ответственным и статусом
  - кнопка `Обновить`
  - отображение общего количества задач и времени обновления
- Создан `.env` с вашим вебхуком:
  - `B24_WEBHOOK_URL=https://b24-ikla82.bitrix24.ru/rest/1/y5n4g5a9f5m7q555/`

Проверки:

- `pnpm install` — успешно
- `pnpm typecheck` — успешно
- `pnpm build` — успешно
- Проверил API запросом к `http://localhost:3001/api/bitrix/tasks/eisenhower` — вернулись задачи, уже корректно разложенные по всем 4 квадрантам.
